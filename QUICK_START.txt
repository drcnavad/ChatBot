ğŸ“ˆ UNDERSTANDING THE OUTPUT:

  ENSEMBLE PREDICTION:
    The final prediction combines all three models using weighted averaging.
    This is generally more accurate than any single model.
  
  CONFIDENCE RANGE:
    Shows the Â± 1 standard deviation range based on model agreement.
    Wider range = models disagree more = less confident prediction
  
  MODEL PREDICTIONS:
    Shows what each individual model predicts:
    - Linear Regression: Simple trend following
    - Random Forest: Captures complex non-linear patterns
    - Gradient Boosting: Learns from sequential errors
  
  VALIDATION METRICS:
    - MAE: Average error in dollars
    - RMSE: Root mean squared error (penalizes big errors)
    - MAPE: Mean absolute percentage error
